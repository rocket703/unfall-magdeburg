# PUV Microsite – Private Unfallversicherung Magdeburg

Eine moderne, responsive Onepage-Microsite zur privaten Unfallversicherung in Magdeburg & Umgebung.  
Sie ergänzt die bestehenden Microsites zur **Zahnzusatz-** und **Berufsunfähigkeitsversicherung**.

---

## Zielsetzung

Einfach verständliche Informationen rund um die private Unfallversicherung –  
klar strukturiert, mit direkter Kontaktmöglichkeit über ein DSGVO-konformes Formular (**Web3Forms**).

## Design-Konzepte

- Modernes, klares Layout im Stil der bestehenden Microsites  
- Lesefreundliche Typografie (*Inter* & *Sora*)  
- Hero-Sektion mit Bildüberlagerung und Call-to-Action  
- Dezente Scroll-Animationen über `.reveal`-Klassen  
- Einheitliches Farb- und Icon-Konzept für alle Versicherungsseiten  

---

## Responsive Umsetzung

- Mobile-first implementiert  
- Flexible Grid-Systeme (`grid-2`, `benefits`, `cta`)  
- Sticky Header mit Burger-Menü  
- Performanceoptimierte Bilddarstellung (WebP-Format)
