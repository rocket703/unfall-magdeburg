PUV Microsite – Private Unfallversicherung Magdeburg

Eine moderne, responsive Onepage-Microsite zur privaten Unfallversicherung in Magdeburg & Umgebung.
Sie ergänzt die bestehenden Microsites zur Zahnzusatz- und Berufsunfähigkeitsversicherung.

Ziel:
Einfach verständliche Informationen rund um die private Unfallversicherung – klar strukturiert, mit direkter Kontaktmöglichkeit über ein DSGVO-konformes Formular (Web3Forms).
